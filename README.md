# Алексей Захарчук — Стихотворения

Личный поэтический сайт. Хостинг: Cloudflare Pages. Репозиторий: GitHub.

---

## Как добавить новое стихотворение

1. Зайди в папку `poems/` в репозитории.
2. Нажми **Add file → Create new file**.
3. Назови файл латиницей, без пробелов, с расширением `.md`.
   Пример: `tishina-v-gorode.md`
4. В начале файла вставь блок с заголовком:

```
---
title: "Тишина в городе"
date: 2024-06-01
---
```

5. Ниже — текст стихотворения. Между строфами — пустая строка.
6. Нажми **Commit changes**.

GitHub Actions автоматически обновит список стихотворений, и через 1–2 минуты стихотворение появится на сайте.

---

## Как изменить существующее стихотворение

1. Открой нужный файл в папке `poems/`.
2. Нажми на иконку карандаша (Edit).
3. Внеси правки.
4. Нажми **Commit changes**.

---

## Как изменить шрифт или цвет

Открой файл `style.css`. В самом начале есть блок `:root { }` с переменными.

- `--gold` — цвет заголовков и акцентов
- `--text-poem` — цвет текста стихотворений
- `--bg-dark` — цвет фона

Меняй значения, сохраняй, сайт обновится автоматически.

---

## Как поменять фоновое изображение

Загрузи изображение в папку `images/`, затем в `style.css` найди строку с `background-image` в блоке `body::before` и измени путь.

---

## Структура репозитория

```
index.html              — главная страница
style.css               — все стили
main.js                 — логика сайта
poems/
  index.json            — список стихов (генерируется автоматически)
  *.md                  — файлы стихотворений
images/
  ...                   — изображения
.github/workflows/
  generate-index.yml    — автоматическое обновление списка
```
